# AssetFlow Database

## Database

- **Database:** MySQL 8.0
- **Database Name:** `assetflow_db`

## Files

- `schema.sql` – Creates the database and tables.
- `seed.sql` – Inserts sample data for demo/testing.

## Setup

1. Run `schema.sql`
2. Run `seed.sql`

## Tables

- departments
- employees
- asset_categories
- assets
- allocations
- transfer_requests
- bookings
- maintenance_requests
- notifications
